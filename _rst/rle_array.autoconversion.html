rle_array.autoconversion module¶

auto_convert_to_rle(df: pandas.core.frame.DataFrame, threshold: float = 1.0) → pandas.core.frame.DataFrame¶

Auto-convert given DataFrame to RLE compressed DataFrame.

Important

Datetime columns are currently not compressed due to pandas not supporting them.

Parameters

  • df – Input DataFrame, may already contain RLE columns.

  • threshold – Compression threshold (1 compresses all, 0.5 compresses if at least 50% memory are safed, 0.0 does not compress at all).
